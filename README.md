Informations:
------------
- Le jeu ne nécessite pas une installation, en revanche vous devez avoir une JVM recente installée sur votre machine pour pouvoir jouer.

- Pour que le jeu puisse fonctionner correctement vous devez impérativement copier tout le dossier <Jeu>, et évitez de supprimer ou de modifier un fichier.

- Pour jouer, double cliquer sur TetrisNouveau.jar
